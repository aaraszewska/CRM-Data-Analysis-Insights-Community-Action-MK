# CRM Data Analysis & Insights – Community Action MK

## 📌 Project Overview
This project analyses CRM data from a community organisation to assess data quality, organisational structure, and digital presence.

## 🎯 Objectives
- Evaluate data completeness and quality
- Analyse organisational sectors
- Assess availability of contact and digital presence data
- Provide actionable recommendations

## 🛠 Tools Used
- Python (Pandas)
- Excel
- Tableau / Power BI

## 📊 Key Findings
- 84% of organisations have an email address
- 77% have a website
- Over 90% of key operational fields (staff, volunteers, beneficiaries) are missing
- Majority of organisations belong to the voluntary sector

## ⚠️ Data Quality Issues
Significant missing data limits the ability to perform deeper analysis and impacts decision-making.

## 💡 Recommendations
- Improve data collection processes
- Enforce mandatory fields
- Standardise CRM input

## 📈 Outcome
Created a dashboard to monitor CRM data quality and support better decision-making.
